Regenerate or add your own:

```bash
python cli.py run --mode workflow --company "Notion"   --out examples/notion_workflow.md
python cli.py run --kind topic  --company "any topic"  --out examples/topic.md
python cli.py run --kind leader --company "a leader"   --out examples/leader.md
python cli.py run --kind party  --company "a party"    --out examples/party.md
```

For the head-to-head cost/latency comparison behind `DECISION_DOC.md`, use:

```bash
python cli.py benchmark --company "Notion"
```

> free-tier 100K-tokens/day cap in a single misbehaving run — see `DECISION_DOC.md` §2.
